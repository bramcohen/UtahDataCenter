This is a utility for recording all writes to an object 
and then playing them back later.

<pre>
>>> from UtahDataCenter import utah
>>> x = utah({})
>>> x[3] = []
>>> x[3].append({})
>>> x[3][0][4] = 'abc'
>>> x
{3: [{4: 'abc'}]}
>>> x.dump()
[3] = []

  File "<stdin>", line 1, in <module>
